HTML is moeilijk

Les 4 html gaat over een probleem die ik ervaar. Dit is de fix.

Het probleem

Waar ik veel last van heb tijdens het code van HTML is het zorgen dat de syntax klopt. ik vergeet vaak divs af te sluiten en daardoor kan een heel bestand soms niet kloppen.

De oplossing!

  1. Maar hoe zorg ik dat in het wel goed doe?

  2. Als je a zegt moet je ook B zeggen!

  3. Snap dat wanneer je een stuk opent zoals een div of een paragraph dat je deze ook weer afsluit

  4. Uiteindelijk word het een gewoonte om als je A zegt ook B te zeggen.

  5. Oefening baart kunst dus let er altijd op wanneer je code

Happy Coding!
